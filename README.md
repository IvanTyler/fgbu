Используемые технологии: Next.js, TypeScrit, scss modules.

Для запуска проект установите зависимости командой npm i, затем npm run dev.

## 🧠 Архитектурные решения

- **Сохранение состояния пагинации:** Номер страницы читается из query-параметра `?page_number=1`. При возврате со страницы поста пагинация восстанавливается.
- **Типизация:** Все компоненты, пропсы и API-ответы строго типизированы (TypeScript).
- **Роутинг:** Динамические страницы постов через `app/posts/[id]/page.tsx`.
[✈️ Живой пример] https://fgbu.vercel.app/?page_number=1
