Используемые технологии: Next.js, TypeScrit, scss modules.

Для запуска проект установите зависимости командой npm i, затем npm run dev.

- Реализован постраничный вывод постов в 10ти количествах на странице.
- Реализована пагинация с возможностью сохранения ссылки на активную страницу.
- Реализована отдельная страница поста, при возврата сохраняется активная страница пагинации.
